Sokratischer Dialog

KI-Sprachmodelle können nicht nur so genutzt werden, dass sie schnell eine fertige Antwort generieren, sondern dass sie zum vertieften Nachdenken herausfordern. Ein sokratischer Dialog bedeutet: Das KI-Modell gibt keine Antworten, sondern stellt gezielte Rückfragen, die dabei helfen, eigene Gedanken zu entwickeln und ein Thema tief zu durchdringen.

Gib ein Thema ein, das du gern tiefer durchdenken möchtest. Zum Beispiel: Nachhaltigkeit in der beruflichen Praxis.



Weitere ähnliche Möglichkeiten

Kopfstand-Methode

Die Kopfstand-Methode ist eine Kreativitätstechnik: Anstatt direkt nach Lösungen zu suchen, überlegt man zunächst, was das Dümmstmögliche wäre, das man tun könnte. Danach fällt es erfahrungsgemäß viel leichter, auf neue und hilfreiche Ideen zu kommen.

Gib eine Herausforderung als Frage ein. Zum Beispiel: Wie können wir Auszubildende besser bei Prüfungsvorbereitungen unterstützen?



Brainstorming-Spiel

Beim Brainstorming-Spiel geht es darum, mit dem KI-Sprachmodell in ein kreatives PingPong zu kommen: Du nennst eine Idee, dann die KI, dann du wieder. So entstehen abwechselnd neue Perspektiven auf eine Herausforderung.

Gib eine Frage ein, zu der du neue Ideen entwickeln willst. Zum Beispiel: Wie kann unser Betrieb digitale Kompetenzen stärken?
